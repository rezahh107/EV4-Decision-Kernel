# Not-applicable dispositions

Only schema-valid `coverage-not-applicable-disposition.v1` JSON artifacts belong here. A file in this directory is not accepted by location or by author assertion: `validate:coverage` derives acceptance from the exact record/link subject, reason code, validator rule/version, evidence head, and a source statement matching the affected record's verified lineage.

No active disposition exists in `DCOV-EXEC-001`. Documentation in this directory cannot create coverage credit.
